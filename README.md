# Sistema de Biblioteca Online

Proyecto web desarrollado con HTML, Bootstrap 5, CSS, JavaScript, PHP y MySQL.

## Roles del sistema

- Administrador: gestiona usuarios y roles.
- Bibliotecario: gestiona libros, préstamos y devoluciones.
- Lector: consulta el catálogo, solicita libros y devuelve libros.

## Base de datos

El archivo `databas.sql` contiene la estructura de la base de datos y datos iniciales.
